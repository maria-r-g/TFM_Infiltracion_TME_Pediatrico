# TFM - Caracterización de tumores sólidos pediátricos según la infiltración del sistema inmune mediante datos transcriptómicos

Repositorio con el código desarrollado para el Trabajo Fin de Máster en Bioinformática y Bioestadística (UOC).

## Descripción

El objetivo del trabajo fue caracterizar y comparar los perfiles de infiltración inmunitaria en tumores sólidos pediátricos mediante datos transcriptómicos públicos obtenidos de cBioPortal.

El análisis se realizó en R e incluye:

- Descarga y preparación de datos.
- Normalización y preprocesado.
- Deconvolución inmune mediante ConsensusTME.
- Análisis de asociación con variables clínicas.
- Análisis de supervivencia mediante Kaplan-Meier.
- Generación de figuras y tablas.
